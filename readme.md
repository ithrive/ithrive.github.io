# iThrive website

iThrive website developed using [Jekyll](jekyllrb.com) and hosted on [GitHub pages](https://pages.github.com/).


## Creating

```
bundle exec jekyll page "My New Page"
```

```
bundle exec jekyll page "My New Page"
```


## Development

Install RVM (see website).

```
rvm install ruby 2.2.2
gem install bundler
```

Install:
```
npm install
bundle install
```

Run:
```
grunt serve
```
